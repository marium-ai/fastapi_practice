# FastAPI Practice 🚀

My Python and FastAPI learning journey — daily practice exercises along with small API projects built while learning backend development.

## 📂 What's Inside

### Daily Practice
A series of daily Python exercises (`day1.py` through `day8.py`) tracking my progress as I learn core concepts and FastAPI fundamentals.

### Mini Projects
- **BMI.py** — Body Mass Index calculator
- **temperature_convertor.py** — Converts temperatures between units
- **cafe.py** — A simple cafe ordering system
- **shop.py** — A basic shop/inventory app
- **library.py** — Library management system
- **patient.py** — Patient record management (uses `patients.json` for data)
- **student_managmentAPI.py** — Student management REST API built with FastAPI
- **task.py** — Task management app
- **lip_gloss.py** — A small project using `lip_gloss.json` for data
- **main.py** — Main entry point for testing/running the FastAPI app
- **ac.py / exp.py** — Additional practice scripts

## 🛠️ Tech Stack
- **Python**
- **FastAPI**
- JSON for lightweight data storage

## 📌 Purpose
This repo is a personal learning log — not a production project. Each file represents a step in understanding how APIs, request/response handling, and Python fundamentals work together.

## 🚧 Status
Actively growing as I learn more FastAPI concepts (path parameters, request bodies, validation, databases, etc.).
